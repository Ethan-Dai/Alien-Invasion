# play
**need pygame support**，run the file 'anlien_invasion.py' to start the game

# KEY:

Q:   Quit the game       
P:   Pause         
C：Change the bullet type           
U：Upgrade the ship            
space：fire (support hold)      
 ↑ ↓ ← → ：move the ship
